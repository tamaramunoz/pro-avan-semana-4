# Programación Avanzada - Semana 4

Este repositorio contiene la solución práctica para la **Semana 4** de la asignatura **Programación Avanzada**. El objetivo de esta semana fue examinar, seleccionar e implementar colecciones de datos típicas en el lenguaje de programación Python (listas, tuplas, conjuntos y diccionarios), utilizando sus atributos y operaciones avanzadas para almacenar, organizar y manipular información de manera estructurada y eficiente.

## Conceptos Técnicos Aplicados

1. **Estructuras Estructuradas Híbridas (Lista de Diccionarios):** Implementación de una lista global dinámica y mutable como contenedor principal para albergar diccionarios anidados, permitiendo modelar de forma óptima productos de la heladería "Frosty Delights" al asociar llaves descriptivas (`sabor`, `precio`, `stock`) a sus respectivos valores de negocio.
2. **Mutabilidad y Operaciones de Expansión:** Uso de métodos nativos de manipulación de colecciones, específicamente `.append()`, para agregar de manera dinámica nuevos elementos al inventario en tiempo de ejecución, demostrando la flexibilidad y el comportamiento mutable de las listas en Python.
3. **Búsqueda Eficiente y Validación de Duplicados:** Empleo de expresiones de comprensión de listas integradas con la función de control lógico `any()`, con el propósito de inspeccionar el contenido de la colección y evitar la inserción de claves repetidas en el catálogo antes de alterar el estado de la estructura.
4. **Formateo Avanzado de Cadenas y Despliegue de Datos:** Uso de especificadores de precisión decimal (`:.1f`) dentro de cadenas formateadas literales (f-strings) durante la iteración de la lista, garantizando una salida en consola ordenada, estandarizada y legible para el usuario final.
